## Belangrijke keuzes

| Keuze | Reden |
|---|---|
| 3 modi in één section met tabs i.p.v. 3 aparte tabs | Compacter; gebruiker kiest één bron, niet meerdere |
| Wit→transparant conversie op signature_pad output | signature_pad geeft witte achtergrond; voor in-PDF embed wil je transparant |
| SVG `<script>`-tags gestript vóór in DOM laden | XSS-defense bij gebruik van user-supplied SVG via Image-tag |
| SVG → PNG rasterize i.p.v. SVG embedden | pdf-lib heeft geen native SVG-support; rasterize via canvas is de simpelste robuuste weg |
| Aspect-ratio behouden bij plaatsing | Slider voor breedte, hoogte automatisch — geen rare stretches |
| Fabric.js geschrapt | Niet meer nodig — pdf-lib drawImage is direct genoeg; vermijdt extra CDN-dep |
| Max-sizes per modus (10MB bitmap, 2MB SVG) | Defensieve grenzen tegen browser-memory-issues |
| signature_pad lazy init bij modus C | Performance + voorkomt fouten als script later laadt |

# prompts/

Sessie-MDs per Meta_Master `Prompt Sessie Documentatie Protocol`.

Naamconventie: `YYYY-MM-DD_korte_slug.md`
Frontmatter: `date / repo / status: open|pending|done / resume: "<trigger>"`
Template: `Meta_Master/templates/PROMPT_SESSION_TEMPLATE.md`

Bij wijziging frontmatter: run `python3 Meta_Master/tools/update_resume.py`.
